HTML Web Design: Review

This is the final project for the Fall 2025 IEA (Institute for Educational Advancement) class; HTML Web Design.

In this class, we learned to create and remix many projects made off of Visual Studio code, like fake business websites, quizzes, and excuse generators.

First Project

Our first project consisted of a simple website. We learned how to use "p" tags (paragraph tags), "strong" tags (bold text), creating lists (ordered and unordered), and styling these elements with CSS, a code language used to style elements and, in a nutshell, make things look nice. I created a sample website, a simple "reasons to sub to my yt channel" page.

We learned to identify parts of a link, what variables (placeholders for information) are, and how to add external links into HTML using the "a" tag (anchor tag), which is used to incorporate links into a program.

Sample Website, BUT BETTER

For our next project, we learned how to incorporate images into HTML, how to add links using the anchor and link tags into HTML, and expanding our CSS capabilities. I copied my "reasons to sub to my yt channel" project and incorporated links, variables, and images into the project, making it interactive and visually appealing.

Eventually, after some practice with CSS and HTML, we moved on to JavaScript, a programming language that addresses how things work behind the scenes, similar to the backstage of a Broadway play.

A FREE JS YouTube Title Generator !!

For our first project utilizing JS, we were assigned to create an excuse generator using JS. This project was made up of buttons, arrays (a set of multiple variables), and functions to create the excuse generator. After I created my excuse generator, I created a spin-off of this project by creating a YouTube Title Generator to help with my Youtube Channel ideas. All I did to make this was restyle the webpage with CSS, and change the values within the arrays that held the words to make the original "excuse".

Conclusion

Overall, this class was extremely fun to participate in! I got to learn a lot of skills with HTML, CSS, and JS, and I'm excited to apply this work to many of my hobbies, like game design, content creation, and more. Thanks IEA!
